## Different types of pattern

Reading is included in all of these skillset somehow.

```Haskell
Aim
  -> Pure Jump
  -> Slider Aim
  -> Aim Control

Aim + Density
  -> Burst
  -> Cut Stream
  -> Slow Stream ( Under/eq 160bpm, 1/4)
  -> Fast Stream ( Above 160bpm, 1/4 )
  -> Outburst ( Under/eq 160bpm, 1/8 )

Additional mods:
  + Novelty -> Instrument/Symphony
  + Intensity -> Hard Rock/7~ stars map
  + Brainstorm -> AR Gimmick/Pure Gimmick
```
