# dnf

### dnf command
- dnf install                     -> abbr: dnf in
- dnf remove                      -> abbr: dnf rm
- dnf update/upgrade              -> abbr: dnf up
- dnf update/upgrade --refresh    -> this command checks all your package if it is up to date.
