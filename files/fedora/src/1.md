# dnf

### dnf usage
```shell
sudo dnf install/remove/... {packages: required} - abbr: sudo dnf in/rm<br>
sudo dnf update/upgrade/... {packages: optional} - abbr: sudo dnf up,<br>
&emsp;&emsp; _--refresh_ flag tells dnf to fetch the newest repository, and upgrade all packages at the same time
```

### /etc/dnf/dnf.conf
```yaml
max\_parallel\_downloads=1<br>
install\_weak\_deps=False<br>
exclude=flameshot
```
