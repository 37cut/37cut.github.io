桌面配置

以{account}身份登录

启动服务

sudo systemctl enable acpid bluetooth tlp NetworkManager
systemctl start NetworkManager
 联网用nmtui

软件下载

sudo pacman -S i3{-wm,lock,status}(其他桌面环境)
network-manager-applet pcmanfm-gtk3 xarchiver dmenu
dunst feh ibus ibus-libpinyin picom flameshot mpv
zsh zsh-{completions,autosuggestions,syntax-highlighting}
chromium alacritty xss-lock
下载yay:
wget https://cutt37.github.io/pkgs/yay-12.1.0-1-x86_64.pkg.tar.zst

vim配置(可选)

sudo pacman -S npm nodejs
vim
:PlugInstall
:CocInstall coc-json coc-html coc-css
:q
sudo ln -sf /home/cutt37/.vim /root/.vim

我的配置(可选)

wget https://cutt37.github.io/pkgs/main.zip

unzip main.zip
Execute bash script as root.
sudo ./configuration.sh

电池状态(可选)

sudo vim /etc/profile
添加/path/to/.config/battery-status.sh &

软件

yay -S slim archlinux-themes-slim(其他登录管理界面) ttf-symbola
{aic94xx,ast,wd719x}-firmware upd72020x-fw
缺失的固件(firmware)会在执行mkinitcpio -P时产生警告.
可用的固件:
mkinitcpio - ArchWiki
通过yay -S {固件}下载,然后再次执行sudo mkinitcpio -P

slim配置

sudo systemctl enable slim

sudo vim /etc/slim.conf
/current_theme to search.
current_theme archlinux-simplyblack

echo i3 > .xinitrc

输入法

若使用fcitx输入法请把所有ibus替换成fcitx
sudo echo INPUT_METHOD=ibus >> /etc/environment
sudo echo XMODIFIERS=@im=ibus >> /etc/environment

GTK主题

Nordic-darker: 下载

Uncompress it.
sudo mv Nordic-darker /usr/share/themes

sudo echo GTK-THEME=Nordic-darker >> /etc/environment
