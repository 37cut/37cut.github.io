osu!下载

1.启用 multilib
vim /etc/pacman.conf
Uncomment [multilib] and mirror

2.软件
sudo pacman -Syu
sudo pacman -S wine wine-mono winetricks lib32-{gst-plugins-base-libs,gnutls,vulkan-intel} --needed

3.配置
winecfg WINEARCH=win32
Select "Windows 2000"

winetricks --gui
Select "Select the default wineprefix"
Select "Install a Windows DLL or component"
Select "dotnet40"

第二第三步骤适用于大多数Windows游戏

4.Install osu
mkdir osu && wine osu\!install.exe
