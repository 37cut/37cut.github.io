Side notes

About wine

wine-stable + winetricks is enough.

Run winecfg to generate '.wine' folder(which stores an windows 'sandbox').

Maybe you want to change the dpi, so go to graphics->screen resolution.

And change it to a prefer value.

Run winetricks dotnet45, and done.

Gimp things

Smudge tool can move the pixels to correct position.

Sometimes Path tool is better than Free select.

Denoise and sharpen things:

Filters->Blur->Selective Gaussian Blur

Filters->Blur->Median Blur

'Ctrl+,' or 'Ctrl+.' Fill bg/fg

Filters->Enhance->Sharpen(Unsharp Mask)
